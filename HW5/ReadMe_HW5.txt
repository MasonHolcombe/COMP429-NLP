## To Run HW5_MasonHolcombe.py ##

1) Navigate to folder of zip extract in terminal
    - Must have 'HW5_MasonHolcombe.py' and 'HW5_Utilities.py' in this folder
    - Must have 'wordsim_similarity_goldstandard.csv' in this folder

2) Install dependencies with 'pip install -r requirements.txt'

3) Run 'python .\HW5_MasonHolcombe.py'
